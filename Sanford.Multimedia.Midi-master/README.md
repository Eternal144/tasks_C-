#### 一、功能概述

该项目引用了`Sanford.Multimedia.Midi`这个midi文件播放库，对该项目进行了自适应处理，并对其功能做了一点的扩展，如添加了播放模式和播放列表，并将功能按钮进行了合并

#### 二、项目特色

1. 实现控件的自适应

   通过记录空间的位置，窗口的缩放比例和递归控件，实现空间随着窗口的缩放而缩放

2. 自由设置背景

   用户可自由上传图片作为程序的背景

3. 选择播放模式

   有三种播放模式可供选择，包括：

   * repeat：循环播放
   * order：顺序播放
   * random：随机播放

4. 操作播放列表

   * 添加
   * 当前播放位置高亮
   * 点击播放

5. 功能按钮的合并

   * play
   * stop
   * loaded

#### 六、结论
